# jinen

##Description

工房じねんwebサイト（WordPress)の改善or改修に向けた開発環境。
WordPress環境の構築作業の自動化も目指す。

## Requirement

環境構築のため
- Dockerのインストール
- Docker-composeのインストール

## Usage

1. Dockerのインストール
	'% brew cask install docker'
2. docker-composeのインストール
	'$ brew install docker-compose'
